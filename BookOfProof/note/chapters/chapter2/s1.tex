\subsection{Statement}
\begin{definition}
    [Statement] A statement is a sentence or a mathematical expression that is either definitely true or definitely false. 
\end{definition}
Followings are some examples of statements:

\begin{example}
    $$2 \in \Z$$
    $$\sqrt{2} \notin \Z$$
    For every integer $n > 1$, the number $2^n - 1$ is prime. 
\end{example}

When a sentence or statement $P$ contains a variable such as $x$, we sometimes denote it as $P(x)$ to indicate that it 
is saying something about $x$.

\begin{example}
    $P(x)$: If an integer $x$ is a multiple of 6, then x is even. 
\end{example}

\begin{definition}
    [Open sentence]
    A sentence such as this, whose true depends on the value of one or more varaibles, is called an open sentence. 
\end{definition}

\begin{definition}
    [Proof]
    A proof is a sequence of of true statements, without logical gaps, there is a logical argument establishing some conclusion. 
\end{definition}
