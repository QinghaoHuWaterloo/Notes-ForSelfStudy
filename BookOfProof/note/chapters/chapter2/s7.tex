\subsection{More on Conditional Statements}
\begin{definition}
    If $P$ and $Q$ are statements are open sentences, then 
    \[
        P \implies Q
    \]
    is a statement. This statement is true if it's impossible for $P$ to be true while $Q$ is false. 
    It is false if there is at least one instance in which $P$ is true and $Q$ is false. 
\end{definition}

\subsubsection*{Facts}
The following statements mean the same thing.

Given element x and Set X which x is an element of Set X, and a statement about x Q(x)
\begin{paracol}{2}
    \begin{leftcolumn}
        \[
        \forall x \in X, Q(x)
        \]
    \end{leftcolumn}
    \begin{rightcolumn}
        \[
        \forall x \in X \implies Q(x)
        \]
    \end{rightcolumn}
\end{paracol}
